## Getting Started

### Prerequisites

- Node.js 18+ installed
- npm or yarn package manager

### Installation

1. **Copy this template to your desired location**:
```bash
cp -r /Users/geetanshgoyal/Desktop/website/devforge-template /Users/geetanshgoyal/Desktop/devforge
cd /Users/geetanshgoyal/Desktop/devforge
```

2. **Install dependencies**:
```bash
npm install
```

3. **Run the development server**:
```bash
npm run dev
```

4. **Open your browser**:
Navigate to [http://localhost:3000](http://localhost:3000)

## Customization

### Update Content

Edit the component files in `/components`:
- `hero.tsx` - Main landing section
- `about.tsx` - About section features
- `events.tsx` - Events and activities
- `projects.tsx` - Club projects
- `team.tsx` - Team members
- `join.tsx` - Contact information
- `footer.tsx` - Footer links

### Update Styling

- Global styles: `app/globals.css`
- Tailwind config: `tailwind.config.ts`

### Update Metadata

Edit `app/layout.tsx` to update:
- Page title
- Description
- Keywords

## Build for Production

```bash
npm run build
npm start
```
